Extreme Value Analysis using Generalized Extreme Value (GEV) Distribution

This Python script performs extreme value analysis using the Generalized Extreme Value (GEV) distribution. The dataset is divided into three sections, and the top three values from each section are selected for analysis.

Steps:

Read Dataset:

The script reads a dataset from a text file ('DATA.txt').
Divide Dataset:

The dataset is divided into three sections of equal size.
Top Values Selection:

The top three values from each section are identified and stored in the 'top_values' list.
GEV Parameter Estimation:

The GEV distribution parameters (shape, location, and scale) are estimated using the 'fit' function from the 'scipy.stats.genextreme' module.
CDF Calculation:

The cumulative distribution function (CDF) values are calculated using the GEV parameters for the selected top values.
Plotting:

A plot of CDF values against the corresponding top values is generated using Matplotlib.
Output:

The script saves the GEV distribution parameters and the CDF values along with corresponding top values to 'output.txt'.
Display Plot:

The CDF plot is displayed and saved as 'CDF_Plot.png'.
Files Generated:

output.txt: Contains GEV parameters and CDF values.
CDF_Plot.png: Visual representation of the cumulative distribution function.
Usage:

Replace 'E:\DATA.txt' with the path to your dataset file.
Ensure the necessary Python libraries (numpy, matplotlib, scipy.stats) are installed.
Note:

This script provides insights into the extreme values of the dataset, helping in understanding the tail behavior using the GEV distribution.
